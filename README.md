# R203 - Qualité de Développement

> Module R203 — IUT Informatique, Université Côte d'Azur (2025)

## Description

Ce dépôt regroupe l'ensemble des supports de cours, travaux pratiques et ressources du module **R203 — Qualité de Développement**. Le module couvre les bonnes pratiques de développement logiciel : clean code, gestion de versions avec Git, et méthodologies de travail collaboratif.

## Contenu des TPs

| TP | Thème | Concepts clés |
|----|-------|---------------|
| **TP1** | Clean Code | Nommage, refactoring, principe DRY, lisibilité |
| **TP2** | Git — Les bases | Clone, add, commit, push, .gitignore, dépôt distant |
| **TP3** | Git — Avancé | git diff, git show, checkout, restore, stash |
